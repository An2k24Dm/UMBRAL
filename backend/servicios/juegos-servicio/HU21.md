# HU21 — Crear Búsqueda del Tesoro

**Como** Operador, **quiero** crear una búsqueda del tesoro en estado Borrador para luego configurarle etapas y misiones antes de activarla.

---

## Criterios de aceptación

- La búsqueda del tesoro se crea en estado **Borrador**.
- El nombre de la búsqueda debe ser único.
- Se registra el evento de integración `BusquedaCreada` en el Outbox (diseño preparatorio).
- El Operador puede listar sus búsquedas en borrador.

---

## Flujo — Crear búsqueda del tesoro

```
Frontend → POST /api/juegos/busquedas
           Authorization: Bearer <token-operador>
           Body: { nombre, descripcion }
        ↓
CrearBusquedaTesoroComando(dto, creadorId) → MediatR
        ↓
CrearBusquedaTesoroManejador:
  1. repositorio.ExisteBusquedaConNombreAsync(nombre)
       → si existe → ExcepcionDominio (422)
  2. BusquedaTesoro.Crear(nombre, descripcion, creadorId, fechaHoraUtc)
       → Estado = Borrador
       → registra BusquedaCreadaEvento
  3. repositorio.CrearBusquedaTesoroAsync(busqueda)
       → INSERT BusquedaTesoro
        ↓
HTTP 201 Created  { id: guid }
```

## Flujo — Listar búsquedas en borrador

```
Frontend → GET /api/juegos/busquedas/borrador
           Authorization: Bearer <token-operador>
        ↓
ObtenerBusquedasEnBorradorConsulta(creadorId) → MediatR
        ↓
ObtenerBusquedasEnBorradorManejador:
  repositorio.ObtenerBusquedasEnBorradorAsync(creadorId)
    → SELECT ... WHERE creador_id = ? AND estado = 0 ORDER BY fecha_creacion DESC
        ↓
HTTP 200 OK  [ BusquedaTesoroResumenDto... ]
```

---

## Endpoints

| Método | Ruta | Descripción |
|--------|------|-------------|
| `POST` | `/api/juegos/busquedas` | Crear búsqueda del tesoro (Borrador) |
| `GET`  | `/api/juegos/busquedas/borrador` | Listar búsquedas en borrador del operador |

---

## Modelo de dominio

| Entidad | Descripción |
|---------|-------------|
| `BusquedaTesoro` | Aggregate Root — juego de búsqueda del tesoro |
| `Etapa` | Fase ordenada dentro de la búsqueda (se agrega en HU22) |
| `Mision` | Tarea dentro de una etapa (se agrega en HU23) |
| `EstadoBusqueda` | `Borrador=0`, `Activa=1`, `Archivada=2` |
| `TipoMision` | `PistaTexto=0`, `Acertijo=1`, `CodigoQR=2` |
