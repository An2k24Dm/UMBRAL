# HU24 — Modificar / Eliminar Etapa

**Como** Operador, **quiero** editar el título y descripción de una etapa, o eliminarla completamente, **para** reestructurar la Búsqueda del Tesoro antes de activarla.

---

## Criterios de aceptación

- Solo se pueden modificar o eliminar etapas de una búsqueda en estado **Borrador**.
- Si la búsqueda está **Archivada**, se retorna 422.
- Si la búsqueda o la etapa no existen, se retorna 404.
- Al eliminar una etapa, sus misiones se eliminan en **cascada** (FK con `ON DELETE CASCADE` en BD).

---

## Flujo — Modificar etapa

```
Frontend → PUT /api/juegos/busquedas/{busquedaId}/etapas/{etapaId}
           Authorization: Bearer <token-operador>
           Body: { nuevoTitulo, nuevaDescripcion }
        ↓
ModificarEtapaComando(busquedaId, etapaId, dto) → MediatR
        ↓
ModificarEtapaManejador:
  1. repositorio.ObtenerBusquedaPorIdAsync(busquedaId)
       → si no existe → ExcepcionNoEncontrado (404)
  2. busqueda.ModificarEtapa(etapaId, nuevoTitulo, nuevaDescripcion)
       → si Archivada → ExcepcionDominio (422)
       → si etapa no encontrada → ExcepcionNoEncontrado (404)
  3. repositorio.ModificarEtapaAsync(busquedaId, etapa)
        ↓
HTTP 204 No Content
```

## Flujo — Eliminar etapa

```
Frontend → DELETE /api/juegos/busquedas/{busquedaId}/etapas/{etapaId}
           Authorization: Bearer <token-operador>
        ↓
EliminarEtapaComando(busquedaId, etapaId) → MediatR
        ↓
EliminarEtapaManejador:
  1. repositorio.ObtenerBusquedaPorIdAsync(busquedaId)
       → si no existe → ExcepcionNoEncontrado (404)
  2. busqueda.EliminarEtapa(etapaId)
       → si Archivada → ExcepcionDominio (422)
       → si etapa no encontrada → ExcepcionNoEncontrado (404)
  3. repositorio.EliminarEtapaAsync(busquedaId, etapaId)
       → DELETE en cascada (Etapa + Misiones)
        ↓
HTTP 204 No Content
```

---

## Endpoints

| Método   | Ruta                                               | Descripción              |
|----------|----------------------------------------------------|--------------------------|
| `PUT`    | `/api/juegos/busquedas/{busquedaId}/etapas/{etapaId}` | Modificar datos de etapa |
| `DELETE` | `/api/juegos/busquedas/{busquedaId}/etapas/{etapaId}` | Eliminar etapa           |
