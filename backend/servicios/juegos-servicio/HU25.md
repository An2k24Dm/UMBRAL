# HU25 — Modificar / Eliminar Misión

**Como** Operador, **quiero** editar el contenido, tipo o pista clave de una misión, o eliminarla de su etapa, **para** ajustar la Búsqueda del Tesoro antes de activarla.

---

## Criterios de aceptación

- Solo se pueden modificar o eliminar misiones de una búsqueda en estado **Borrador**.
- Si la búsqueda está **Archivada**, se retorna 422.
- Si la búsqueda, etapa o misión no existen, se retorna 404.

---

## Flujo — Modificar misión

```
Frontend → PUT /api/juegos/busquedas/{busquedaId}/etapas/{etapaId}/misiones/{misionId}
           Authorization: Bearer <token-operador>
           Body: { nuevoTitulo, nuevaDescripcion, nuevoTipo, nuevaPistaClave }
        ↓
ModificarMisionComando(busquedaId, etapaId, misionId, dto) → MediatR
        ↓
ModificarMisionManejador:
  1. repositorio.ObtenerBusquedaPorIdAsync(busquedaId)
       → si no existe → ExcepcionNoEncontrado (404)
  2. busqueda.ModificarMision(etapaId, misionId, ...)
       → si Archivada → ExcepcionDominio (422)
       → si etapa/misión no encontrada → ExcepcionNoEncontrado (404)
  3. repositorio.ModificarMisionAsync(etapaId, mision)
        ↓
HTTP 204 No Content
```

## Flujo — Eliminar misión

```
Frontend → DELETE /api/juegos/busquedas/{busquedaId}/etapas/{etapaId}/misiones/{misionId}
           Authorization: Bearer <token-operador>
        ↓
EliminarMisionComando(busquedaId, etapaId, misionId) → MediatR
        ↓
EliminarMisionManejador:
  1. repositorio.ObtenerBusquedaPorIdAsync(busquedaId)
       → si no existe → ExcepcionNoEncontrado (404)
  2. busqueda.EliminarMision(etapaId, misionId)
       → si Archivada → ExcepcionDominio (422)
       → si etapa/misión no encontrada → ExcepcionNoEncontrado (404)
  3. repositorio.EliminarMisionAsync(etapaId, misionId)
        ↓
HTTP 204 No Content
```

---

## Endpoints

| Método   | Ruta                                                                    | Descripción            |
|----------|-------------------------------------------------------------------------|------------------------|
| `PUT`    | `/api/juegos/busquedas/{busquedaId}/etapas/{etapaId}/misiones/{misionId}` | Modificar misión       |
| `DELETE` | `/api/juegos/busquedas/{busquedaId}/etapas/{etapaId}/misiones/{misionId}` | Eliminar misión        |

---

## Tipos de misión (TipoMision)

| Valor | Nombre      |
|-------|-------------|
| `0`   | PistaTexto  |
| `1`   | Acertijo    |
| `2`   | CodigoQR    |
